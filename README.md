# RoboBob

RoboBob is a childrenâ€™s mathematics robot app that interacts with users to answer basic questions about itself and evaluate arithmetic expressions. This project is implemented using Java and Spring Boot, and it includes a REST API to handle user queries.

## Getting Started

### Dependencies

* Java 17 or higher
* Maven

### Installing

* Git clone https://github.com/Mungee/RobobobRepo.git

### Executing program

* Move into to root directory of the project
```
mvn spring-boot:run
```
